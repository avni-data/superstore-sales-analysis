# superstore-sales-analysis
Retail Sales & Profitability Analysis using python

# Project Overview
This project analyzes retail sales transaction data to uncover revenue trends, category-level performance, seasonal demand patterns, and profitability insights using Python.

# Dataset Information
  9,800 transactions
  18 variables
  Includes: Sales, Category, Region, Segment, Order Date, Estimated Profit

# Key Analysis Performed
  Sales distribution analysis
  Category-wise revenue comparison
  Regional performance evaluation
  Seasonal trend analysis (Heatmap)
  Correlation analysis
  Linear regression modeling

# Model Performance
  R² Score: 1.0
  Mean Squared Error: Near zero
  Perfect linear relationship between Sales and Estimated Profit

# Key Business Insights
  Technology category generates highest revenue
  West region shows strongest performance
  Q4 months record peak sales
  Consumer segment drives most revenue
  Profit increases proportionally with Sales

# Tools Used
  Python
  Pandas
  Matplotlib
  Seaborn
  Scikit-learn
  Statsmodels
